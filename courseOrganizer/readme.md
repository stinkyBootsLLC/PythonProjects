# My Course Organizer

Simple script to make new course structure for myself when I take a new course.

i.e.
current directory 

    CMSC405
    
        WEEK1
        
        ....READ
        
        ....ASSIGNMENTS
                assignment_1.docx
                
        ....LAB
                lab_1.docx
                
        ....DISCUSSIONS
                disscusion_1.docx
        WEEK2
        .
        . to
        .
        WEEK(n)

 Course name and weeks will be a user input.

## Installation

- Install Course Organizer by running:

    `git clone courseOrganizer`
    
- Run Script

    `python newCourse.py`
